ClienteFácil – Sistema de Cadastro de Clientes

O ClienteFácil é um sistema simples desenvolvido em Python para gerenciar o cadastro de clientes de um comércio.

Ele permite cadastrar, visualizar, atualizar e excluir clientes de forma prática através de uma interface gráfica.

🔌 Como fazer funcionar na sua máquina

Siga os passos abaixo:

Instale o Python (versão 3 ou superior)
Baixe ou clone este repositório:
git clone https://github.com/RafaellaBressan/PROGRAMACAO-EM-PYTHON-SENAI-60H/blob/4542e97b7c91c5793ac062035fcca12a96ce9a64/aula_20/main.py
Acesse a pasta do projeto:
cd aula_20
Execute o arquivo:
python main.py
📋 Pré-requisitos do sistema
Python instalado
Sistema operacional (Windows, Linux ou Mac)
Interface gráfica habilitada

💡 Não precisa instalar banco de dados, pois o SQLite já vem com o Python.

🛠️ Tecnologias utilizadas
Python
SQLite (banco de dados)
Tkinter (interface gráfica)
VS Code (editor de código)
⚙️ Funcionalidades do sistema
✅ Cadastro de clientes
✅ Listagem de clientes
✅ Atualização de dados
✅ Exclusão de clientes
💡 Como o sistema funciona
O usuário preenche os dados (nome, email, telefone, endereço)
Clica no botão Cadastrar
O sistema salva no banco de dados
Os dados aparecem na tabela
É possível atualizar ou deletar registros
📌 Versões
Python 3.x
SQLite 3 (padrão do Python)
